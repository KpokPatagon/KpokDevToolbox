KpokEntityTool

KpokEntityTool is a Visual Studio Extension that implement four custom editors allowing a developer to edit entity, data service, table and factory information based on the KpokEntityTool.Models.

Once installed this extension will edit the following files:

Extension Kind Description
*.ent Entity An entity file configures application domain objects.
*.datasvc Data service A data service file configures a class that access databases and allows CRUD operations of Entities as well as filter information from the underlaying repository.
*.tbl Table A table files configures database tables allowing table documentation and script generation.
*.fac Factory A factory file configures application context object for the Inetdev Framework which provides IoC capabilities for application development.

For more information on the extension, see the Getting Started guide.
