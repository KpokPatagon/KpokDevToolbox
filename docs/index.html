    This project contains tools, templates and extensions for developers that helps them to easyly build applications by generating code that is boring to develop or too repetitive.

    Templates

    T4 templates for code generation can be found within the template directory of this documentation repo.

    Available templates

    • DataModel-1.0.tt
      Generates domain object models based on entities (*.ent) models.
    • DataModelWithInputModel-1.0.tt
      Generates domain object models based on entities (*.ent) models and an extra simple input model class with only the data properties, that can be used in Asp.Net controllers as input.
    • DataServices-1.0.tt
      Generates data services based on data service (*.datasvc) models.
    • Factory-1.0.tt
      Generates factory configuration XML files based en factory (*.fac) models.
    • MultiTenantDataModel-1.0.tt
      Generates multi-tenant domain object models based on entities (*.ent) models.
    • MultiTenantDataModelWithInputMode-1.0.tt
      Generates multi-tenant domain object models based on entities (*.ent) models and an extra simple input model class with only the data properties, that can be used in Asp.Net controllers as input.
    • MultiTenantDataServices-1.0.tt
      Generates multi-tenant data services based on data service (*.datasvc) models.
